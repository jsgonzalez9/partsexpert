# Product Requirements Document: Parts Compatibility Engine ## Aesthetic: Industrial Minimalist * **Vibe:** Linear.app (slick, fast, command-driven) meets McMaster-Carr (functional, dense, technical). * **Palette:** Neutral grays, stark whites, industrial accents (e.g., #F97316 orange or #2563EB blue). * **Typography:** Geist or Inter for maximum legibility of part numbers and dimensions. ## Core Features 1. **Command-K Search:** A global, lightning-fast search bar as the primary entry point. 2. **Compatibility Checker:** A "Thumb-Friendly" interface for comparing two or more parts for fitment. 3. **Technical Specs:** Dense but readable data tables using shadcn/ui patterns. 4. **Mobile-First Design:** Large touch targets for users in the field/garage. ## Tech Stack (Conceptual) * Next.js 15, Tailwind CSS, Framer Motion, shadcn/ui.
